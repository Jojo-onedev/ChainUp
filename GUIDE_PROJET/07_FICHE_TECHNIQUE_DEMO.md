# 🚀 Fiche Technique de Poche — DiploChain

Garde ces points sous les yeux pendant les 7 minutes de présentation.

## 1. L'Architecture (À mentionner au jury)
- **Réseau** : Polygon Amoy (Layer 2 d'Ethereum).
- **Smart Contract** : Solidity 0.8.24 (Standard, sécurisé, optimisé en gaz).
- **Frontend** : React 19 + Vite (Ultra rapide au chargement).
- **Mobile** : React Native (Expo) pour une portabilité totale iOS/Android.

## 2. Le Point Fort Technique
- **Preuve d'Existence** : "Nous ne stockons pas le diplôme, mais sa preuve cryptographique."
- **Zéro-Friction** : "L'application mobile permet une vérification sans portefeuille crypto pour l'utilisateur final."

## 3. Démo Live : Le "Check-list"
- [ ] Présenter la Landing Page (Branding & Confiance).
- [ ] Émission d'un diplôme (Montrer la simulation des étapes blockchain).
- [ ] Vérification publique (Utiliser le QR Code ou le Hash).
- [ ] Scan Mobile (Montrer l'icône de succès verte).

## 4. Réponses aux questions "Pièges"
- **"Et si l'université se trompe ?"** : On peut révoquer un certificat dans le contrat (fonction non implémentée mais prévue dans la roadmap).
- **"Coût par diplôme ?"** : Moins de 0.01 MATIC (quelques centimes de FCFA).
- **"Stockage des fichiers ?"** : "Les métadonnées peuvent être sur IPFS (décentralisé) pour une sécurité maximale."
